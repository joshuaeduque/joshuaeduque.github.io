MAPILIB

MAPILib is a remote communications library for PS3 consoles. Written in C#, it utilizes sockets to send and receive data to the webMAN MOD server that comes stock with every modern custom firmware. This means that developers can easily create and distribute ready-to-use applications to users using MAPILib.

  • READ / WRITE MEMORY
  • LOAD / UNLOAD PLUGINS
  • SYSCALLS
  • FW VERSION / TEMP / ETC.

MEMORY EDITOR

Immediately after developing MAPILib, I developed a reusable WinForms memory editor control. Like a tradition hex editor, the control allows users to graphically edit bytes in memory as well as display their ASCII values.
